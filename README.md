## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

## 📖 Overview

AI-Powered Crop Health & Pest Risk Monitoring System is a smart agriculture platform that uses Artificial Intelligence and Computer Vision to analyze crop images, detect plant diseases, identify pest infestations, and provide real-time crop health assessment. The system helps farmers make timely decisions, reduce crop loss, and improve agricultural productivity.

---

## ✨ Features

- 🌿 Crop health monitoring
- 🦠 Plant disease detection
- 🐛 Pest risk prediction
- 📷 Image-based crop analysis
- 📊 AI-powered health scoring
- 🚨 Early warning notifications
- 📄 Automated analysis reports
- 📱 Farmer-friendly dashboard
